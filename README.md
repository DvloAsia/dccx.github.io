# DccX Website

## Bilingual Website Structure
- `index.html` - Main language selection page
- `zh/` - Chinese language pages
- `en/` - English language pages
- `assets/` - Shared resources (CSS, JS, images)

## Features
1. **Bilingual Support** - Chinese and English versions
2. **Responsive Design** - Mobile, tablet, desktop friendly
3. **Modern UI** - Clean and professional design
4. **Language Switching** - Easy switch between languages

## GitHub Pages Setup
This website uses GitHub Pages with Jekyll configuration.

## Deployment
Automatically deployed via GitHub Actions when changes are pushed to the repository.
